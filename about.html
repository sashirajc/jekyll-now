ADEX Data Visualization

To protect client's data, only dummy data is used to showcase the project.
ADEX is an exhibition on diving, held in Singapore. The CMS is a part of the app package which allows ADEX admin to edit information on the android/iOS apps. User data collected during the exhibition is stored on MySQL database. This data is displayed with an appropriate visualization for client to see and understand the data better. Most of the charts are interactive. The heatmap can be viewed by date and time with a smooth UI.

Key Insights from the visualizations:

  1. The screen data section has 3 levels of drilldown to show user interest in different sections of the app.
  2. The piecharts show interesting data about the type of users (nationality,divers,age group, gender etc).
  3. The heatmap is useful to see crowd density across the exhibition hall during the event.
  4. Heatmap can further be explored based on day and time of the event.

TKN Game backend Data Visualization

Sansesan is the distributor for TaoKaeNoi,a popular seaweed based snack, in Singapore. The game and CMS is a part of their marketing campaign which ran from Dec 2016 to Febryary 2017. The backend database collected data on the game players which are visualised.
The visualizations help in better understanding the profile of the game players as well as monitor the success of the marketing campaign based on number of games played everyday and number of new users playing the game. The trend can be observed over longer time frames.

Key Insights from the visualizations:

  1. The charts are zoomable which allows for inspection of smaller timeframes.
  2. The piecharts show interesting data about the type of users (age group, gender etc).
  3. The time of day chart gives insights into the most popular time for players to play the game. This can be useful to better target facebook posts at appropriate times.
  4. The last chart shows that most players only played once with the exception of a handful of players who played the game obsessively.
